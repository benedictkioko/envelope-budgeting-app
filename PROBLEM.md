# Envelope Budgeting App

Build a simple envelope budgeting app.

## Features

Users should be able to:

- View a list of envelopes
- Add income
- Move income into an envelope
- Spend money from an envelope

## Rules

- Total available income cannot go below zero
- Envelope balances cannot go below zero

## Requirements

- Backend API
- Frontend UI
- No database required (in-memory is fine)

## Non-requirements

- Authentication
- Styling
- Persistence
- Tests
